初始化
It's Ruff
1. 通过 USB 线缆连接开发板和开发电脑
2. 安装驱动
  • Windows 10: 通过 windows update 自动更新驱动。
  • Windows 7: 下载驱动, http://www.prolific.com.tw/UserFiles/files/PL2303_Prolific_DriverInstaller_v1_12_0.zip
3. 确认端口
  1. 开始→运行→devmgmt.msc
  2. 在设备管理器，查看 端口（COM 和 LPT）
  3. 确认Prolific USB-to-Serial Common Port 对应的 COM 端口。
  4. 点击刷新获取串口列表
